- из репозитория выкачивать только ветку final_version

- для серверной части я использовал фреймворк Codeigniter (т.к. с ним я уже был более-менее знаком).
Файлы проекта находятся в application/controllers, application/models, application/views

- создание необходимых mysql таблиц в файле createTables.sql (сгенерировано через PMA)

- весь js в корне сайта в папке js. Без использования mvc-фреймворков, только js+jquery

- bootstrap && jquery - через CDN

- localhost откроет страницу со списком компаний (комментариев на этой странице нет)

- "вход" производится просто по логину (без пароля), просто для того чтобы идертифицировать разных пользователей.
Сейчас в базе есть 3 пользователя для тестирования "Ivan", "Sergey", "Tom" и несколько компаний. Добавил всех их в базу вручную.

- клик по компании открывает страницу этой компании. На этой странице можно добавлять комментарии к полям (значок справа от каждого из полей). Добавление/просмотр комментариев не доступно незалогиненному юзеру (!).
